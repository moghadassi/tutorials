A list of the most commonly used TCP/UDP ports and their services.

|------|----------|-------------------|---------------------------------------|

| 53   | TCP/UDP  |   DNS             | Domain Name System |
| 67   |   UDP    |   DHCP Server     | Dynamic Host Configuration Protocol |
| 68   |   UDP    |   DHCP Client     | DHCP Client |
| 69   |   UDP    |   TFTP            | Trivial FTP |
| 80   |   TCP    |   HTTP            | HyperText Transfer Protocol |
| 110  |   TCP    |   POP3            | Email Retrieval |
| 123  |   UDP    |   NTP             | Network Time Protocol |
| 135  |   TCP    |   RPC             | Microsoft RPC |
| 137  |   UDP    |   NetBIOS Name Service | Windows Networking |
| 138  |   UDP    |   NetBIOS Datagram | Windows Networking |
| 139  |   TCP    |   NetBIOS Session | Windows File Sharing |
| 143  |   TCP    |   IMAP            | Email Retrieval |

| 1433 |   TCP    |   Microsoft SQL Server | Database |
| 1521 |   TCP    |   Oracle Database | Database |
| 2049 |  TCP/UDP |   NFS             | Network File System |
| 3306 |   TCP    |   MySQL           | Database |
| 3389 |   TCP    |   RDP             | Remote Desktop |
| 5060 |  TCP/UDP |   SIP             | VoIP |
| 5432 |   TCP    |   PostgreSQL      | Database |

| 9200 |   TCP    |   Elasticsearch   | Search Engine |
| 27017 |  TCP    |   MongoDB         | NoSQL Database |

## Notes

- **Well-Known Ports:** 0–1023
- **Registered Ports:** 1024–49151
- **Dynamic/Private Ports:** 49152–65535
